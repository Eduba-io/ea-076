# Coinscrap Finance landing page

Single-file static site prepared for Coinscrap Finance ahead of eMerge Americas Miami 2026 (booth SS81). The page carries a pitch body framed in the prospect's voice and brand tokens (navy #193069, coral #FC7753, Poppins), echoes Coinscrap's signature curly-brace typographic device, and ships a Spanish-primary layout with an EN toggle in the top nav. Eduba appears only in the footer and inside the credibility line.

Intended URL path: /for/coinscrap-finance

## Files

- `index.html`: page markup, Spanish primary with English toggle (small inline script).
- `styles.css`: all styles, Poppins via Google Fonts (same family used on coinscrapfinance.com).
- `logo-coinscrap-white.svg`: white-variant logo copied from the brand pack. Rendered on the navy top bar, brand-accurate.
